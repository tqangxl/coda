## 每日流程

### 晨间例程（6:00-9:00）

```
🌅 晨间唤醒程序

⏰ 6:00-6:15 起床仪式
├── 拉开窗帘，迎接阳光
├── 深呼吸3次
└── 给自己一个微笑 😊

⏰ 6:15-6:45 晨间运动
├── 选择一项运动（跑步/瑜伽/太极）
├── 运动15-30分钟
└── 运动后拉伸5分钟

⏰ 6:45-7:15 个人护理
├── 洗漱
├── 早餐
└── 整理仪容

⏰ 7:15-8:00 专注时间（可选）
├── 晨间阅读/学习
├── 计划今日任务
└── 重要工作启动

⏰ 8:00 出发/开始工作
└── 开启新的一天！
```

### 日间工作（9:00-18:00）

```
💼 日间工作节奏

⏰ 9:00-9:30 启动时间
├── 检查今日日程
├── 处理邮件（限时30分钟）
└── 确认今日重点任务（≤3项）

⏰ 9:30-12:00 高效工作块
├── 关闭通知
├── 专注处理最重要任务
├── 使用番茄工作法（25分钟专注+5分钟休息）
└── 完成2-3个番茄钟

⏰ 12:00-13:00 午休时间
├── 午餐（专注品尝，不看手机）
├── 适度休息（15-20分钟小憩）
└── 外出散步（可选）

⏰ 13:00-14:00 决策时间
├── 下午任务规划
├── 处理需要决策的事项
└── 回复重要消息

⏰ 14:00-17:30 协作工作块
├── 开会/讨论
├── 协作任务
├── 处理日常事务
└── 保持适度休息

⏰ 17:30-18:00 收尾时间
├── 整理今日工作
├── 更新任务状态
├── 准备明日待办
└── 关闭工作环境
```

## 番茄工作法

```
🍅 番茄工作法完整指南

📖 什么是番茄钟？
一个番茄钟 = 25分钟专注工作 + 5分钟休息

🎯 如何使用？

第一步：选择任务
├── 从待办列表中选择一项任务
├── 任务应该能在1-4个番茄钟内完成
└── 如果任务太大，先分解

第二步：设定计时器
├── 设定25分钟
├── 开始计时
└── 全力以赴！

第三步：专注工作
├── 只做这一件事
├── 不看手机
├── 不回消息
└── 不被打断

第四步：休息5分钟
├── 站起来活动
├── 喝水/上厕所
├── 让眼睛休息
└── 不要看手机！

第五步：循环
├── 每完成4个番茄钟
├── 休息15-30分钟
└── 然后继续

📊 每日记录
- 今日番茄数：{count}
- 完成的任务：{tasks}
- 效率评估：{rating}/5

💡 小技巧
- 番茄钟期间，灵感来了？先记下来，继续专注
- 被打断了？重新开始这个番茄钟
- 任务完成了但时间没到？继续检查，不要浪费
```
