## 学习方法指导

### 费曼学习法应用

```
📖 费曼学习法 - 教你用"教"来学

步骤1：选择一个概念
→ 选择你想要理解的知识，比如"分数的加减法"

步骤2：像教小孩一样解释它
→ 用简单的语言写下解释，就像教一个8岁小朋友

步骤3：回顾和简化
→ 哪里卡住了？哪里解释得不够简单？
→ 重新组织语言，用更简单的比喻

步骤4：（可选）传授给他人
→ 把你学到的讲给爸爸妈妈或同学听

示例：分数加法
✗ 复杂版："分子分母不变，分子相加，分母不变"
✓ 简单版："就像切Pizza，分数就是切好的Pizza块"
          "1/4 + 1/4 就是两块1/4的Pizza合起来"
          "合起来就是2块1/4，也就是1/2"
```

### 番茄工作法

```
🍅 番茄工作法 - 提高专注力

准备工具：
- 一个计时器（手机也可以）
- 一张任务纸
- 一支笔

操作步骤：
1. 选择任务：写下今天要完成的任务
2. 设置25分钟：这是你的"番茄钟"
3. 专注工作：只做这一件事！
4. 休息5分钟：站起来动一动，喝点水
5. 重复4次后：休息15-30分钟

小贴士：
- 不要被手机打扰
- 如果走神了，在纸上画个小记号，然后回到任务
- 完成4个番茄钟后，可以奖励自己！
```
